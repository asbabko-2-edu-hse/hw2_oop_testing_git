## Описание проекта
Проект представляет систему управления рецептами и ингредиентами. 
Реализованы функции:
- Создания записи рецепта с возможностью редактирования перечня ингредиентов и категории рецепта
- Создания перечня ингредиентов для рецепта с указанием количества и единицы измерения с возможностью добавления и редактирования, а также с функцией контроля уникальных ингредиентов и автоматическим объединением количества неуникальных
- Автоматического расчета количества требуемых ингредиентов в зависимости от указанного коэффицента
- Создания списка покупок, состоящего из ингредиентов добавленного перечня рецептов, с возможностью редактирования перечня рецептов
- Создания итогового списка покупок путем добавления уникальных ингредиентов из перечня рецептов и неуникальных ингредиентов после объединения их количества с использованием автоматического расчета 

## Использование
### Запуск проекта
1. Клонировать репозиторий
```bash
git clone https://github.com/asbabko-2-edu-hse/hw2_oop_testing_git.git
cd hw2_oop_testing_git
```

2. Установить зависимости
```bash
pip install -r requirements.txt
```

### Запуск тестов
```bash
pytest
```

### Пример запуска кода
```python
from src.Recipe import Recipe
from src.Ingredient import Ingredient

recipe = Recipe("Пицца")
recipe.add_ingredient(Ingredient("Мука", 500, "г"))
print(recipe)
```
